Hilfe / Kurzanleitung

Auf dieser Seite soll kurz die Grundidee der Parkplatzbörse erklärt werden.

  • Wenn jemand einen Parkplatz der Börse zur Verfügung stellen möchte, können unter Meine Parkplätze ein oder mehrere Parkplätze hinzugefügt werden.
  • Ein Parkplatz gehört grundsätzlich seinem Besitzer/seiner Besitzerin.
  • Pro Tag kann der Parkplatz freigegeben werden. Jeder andere Nutzer/jede andere Nutzerin der Börse kann diesen freigegebenen Platz dann buchen, sofern er/sie an diesem Tag noch keinen anderen Parkplatz gebucht hat.
  • Der Besitzer/die Besitzerin eines Parkplatzes kann jederzeit die Freigabe zurückziehen. Das kann sowohl dann passieren, wenn der Parkplatz als 'buchbar' in der Übersicht angezeigt wird, aber auch, wenn ein anderer Nutzer/eine andere Nutzerin der Börse den Parkplatz bereits gebucht hat. In diesem Fall wird der Ausgangszustand wiederhergestellt und sowohl die Freigabe wie auch die Buchung werden entsprechend gelöscht. Aktuell wird über diesen Vorgang noch nicht informiert - bitte selbsständig der Nutzerin/dem Nutzer Bescheid geben.
  • Es können keine Buchungen für die Vergangenheit vorgenommen werden.

Was bedeuten die Symbole?

Auf der Übersichtsseite tauchen je nach Zustand eines Parkplatzes verschiedene Buttons auf, die insgesamt folgende Bedeutung haben:

Symbol / Button Bedeutung
Freigaben Der Besitzer/die Besitzerin eines Parkplatzes kann pro Tag diesen freigeben.
Freigabe Eine erteilte Freigabe für einen Parkplatz kann von der Besitzerin/dem Besitzer wieder gelöscht werden. In diesem Fall war der Parkplatz noch nicht durch eine andere Nutzerin/einen anderen Nutzer gebucht.
Buchen Ein freigegebener Parkplatz kann gebucht werden.
Buchen Ein freigegebener Parkplatz kann nicht gebucht werden, wenn er in der Vergangenheit liegt.
löschen Wenn ich einen Parkplatz gebucht habe, kann ich die Buchung so löschen. Der Parkplatz kann dann wieder von allen anderen Nutzerinnen und Nutzern der Börse gebucht werden, so er/sie an diesem Tag noch keinen anderen Parkplatz nutzt.
USERNAME Wenn ein Parkplatz von einer Nutzerin/einem Nutzer gebucht wurde, hat die Besitzerin/der Besitzer auch dann noch die Möglichkeit den Parkplatz zurückzufordern.

Wo gibt es den Quellcode?

Zukünftige Features
